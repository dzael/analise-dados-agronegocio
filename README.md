# Análise de Dados no Agronegócio

Projeto de análise de dados utilizando Python com foco na interpretação de informações de produção agrícola.

# Objetivo
Analisar dados de produção de soja e milho para identificar padrões e tendências ao longo dos anos.

# Tecnologias
- Python
- Pandas
- Matplotlib
- Google Colab

# Resultados
O projeto permitiu visualizar o crescimento da produção e comparar a produtividade entre culturas.
